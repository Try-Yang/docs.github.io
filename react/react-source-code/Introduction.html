# React源码概览

react官网中的描述，React采用的是monorepo的管理方式，几乎所有的功能都被分为一个个独立的包，来极大的提升联调的效果。

react整体分为react和react-dom两大部分，由以下几个模块组成

# React Core

react的全局API，在react这个模块包中，React只负责去定义组件所需要的API

# Render（渲染器）

负责将ReactDom树，根据平台的不同，渲染城对应平台的UI，核心代码位于react-dom中

# Fiber reconcilers

  • 能够把可中断的任务切片处理。
  • 能够调整优先级，重置并复用任务。
  • 能够在父元素与子元素之间交错处理，以支持 React 中的布局。
  • 能够在 render() 中返回多个元素。
  • 更好地支持错误边界。
